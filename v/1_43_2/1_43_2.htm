V1.43.2 Release Notes

GoTechnology hub2

 

1.   About this Document

Purpose

This document provides a brief overview of the changes made between V1.43 and V1.43.2 of GoTechnology hub2, along with where further details of each change can be found within the updated functional specification, in addition to a listing of completed bug fixes and known issues.

2.   Fixed Issues

The following issues have been fixed in version 1.43.2 of hub2

Case

Title [sic]

Details

93250, 93253, 93257, 93266, 93284, 93323

Custom Fields missing from Search Filters

Custom Fields were not appearing as Search Filters on the Search pages for some entities. These have been re-instated.

93319, 93322, 93327

Internal Server Error when generating document

If a digital document contained a Custom Field bookmark in the Header Table section, an Internal Server Error appeared either when generating the PDF, or when clicking on the Edit Document button.

This error will now no longer appear, and the PDF will be generated (or Edit Document page loaded). The documents may also be downloaded successfully to the Companion App. However, the bookmark will not be populated with a value in either hub2 or the App. This will be resolved fully in a future release.
