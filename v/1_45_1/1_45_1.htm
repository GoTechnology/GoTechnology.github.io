98880

Incorrect warning message in Handover imports

If a Handover had outstanding items and it was updated through the imports, a warning message saying “There are outstanding {Tag ITRs} to be completed before generating the handover” was shown in the import results file. The import could not proceed beyond this point, so the handover could not be updated through the imports.

This has been corrected so that the warning message is no longer displayed in the results file and the import can proceed successfully.
